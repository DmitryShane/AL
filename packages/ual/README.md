# UAL

UAL is the Unity Activity Logger plugin for the AL platform.

## Local MVP Setup

1. Start the AL backend at `http://127.0.0.1:8000`.
2. Import this package through Unity Package Manager.
3. Open `Project Settings > UAL`.
4. Confirm the server URL, author and project id.

The plugin sends encrypted `ALR1` reports to the backend. It does not write local report files.
